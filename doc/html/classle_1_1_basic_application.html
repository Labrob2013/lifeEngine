Базовый класс приложения

Данный класс необходим для работы приложения. Он содержит все необходимые объекты/методы которые нужны, например, для добовления на сцену какой-то модели

Предупреждения
Необходимо определить Update() (В этом методе проходит обновление приложения, например, загрузка карты, движение персонажей и т.д)
