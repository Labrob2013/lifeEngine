Добавить направленый источник света

Этот метод создает направленый источник по переданным характеристикам. По умолчанию источник статичный, чтобы сделать его динамичным надо получить его объект у менеджера и вызвать метод SetDinamicLight (у статичного света не обновляются карты теней)

Аргументы
[in]NameLightНазвание света
[in]PositionНаправление света
[in]ColorЦвет света
[in]IntensivityИнтенсивность света
[in]SpecularОтражение света
